TicMsg가 다른 이유

서버는 시그널링만, 채팅은 직접 P2P

🔑

장기 식별키 (E2EE)

클라이언트 최초 실행 시 ECDH P-256 장기 키쌍이 생성됩니다. 공개키는 서버에 등록, 개인키는 절대 기기 밖으로 나가지 않습니다.

🌐

그리드넷 메시

클라이언트들이 직접 서로 연결됩니다. 가장 가까운 피어를 통해 메시지를 라우팅하여 서버 없이도 채팅이 동작합니다.

🏠

디스코드 스타일 방/채널

방 비밀키만 알면 어떤 기기에서도 입장 가능. 방 안에 여러 채널을 소분류하여 주제별로 대화할 수 있습니다.

💾

선택적 영구 저장

방 설정에서 영구 저장 활성화 시, 오프라인 상태에서도 재접속 시 메시지를 복구할 수 있습니다. 암호문만 서버에 보관됩니다.

📱

최대 5대 기기

하나의 계정으로 최대 5대의 기기를 등록할 수 있습니다. 기기별 공개키를 서버에 등록하여 어떤 기기에서든 인증 가능.

🔒

해시 우선 보안

비밀번호는 bcrypt, 방 키·활성화 키는 SHA-256 해시만 서버에 저장됩니다. 서버가 털려도 원본 키는 알 수 없습니다.

시작 방법

4단계로 안전한 메신저를 시작하세요

1

클라이언트 설치

TicMsg 클라이언트를 설치하면 장기 ECDH 키쌍이 자동 생성됩니다.

2

홈페이지 가입

아이디, 비밀번호와 클라이언트의 공개키를 붙여넣어 가입합니다.

3

활성화 키 발급

대시보드에서 활성화 키를 발급받아 클라이언트에 입력합니다.

4

채팅 시작!

방을 만들거나 참가하여 완전 암호화된 P2P 채팅을 즐기세요.
